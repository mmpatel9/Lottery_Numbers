# Lottery Number Analysis

This project explores historical New York Lottery results using visual analytics to identify trends and patterns in the data. The dataset was cleaned and prepared using Python, with visualisations developed in Tableau to highlight number frequencies, temporal trends, and distribution patterns.

While lottery outcomes are inherently random, this analysis focuses on uncovering interesting structures and recurring behaviours in past draws, presenting them through clear and interactive visual insights.
